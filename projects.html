Cryptum

Winner of Boilermake X Hackathon best security project

In a world where data is the most valuable commodity, a person's information is constantly at risk. Commercial solutions often sacrifice security in the name of profit, and this negligence trickles down to users in negative ways. We wanted to create Cryptum, a file storage app that does better by utilizing the Tillitis TKey technology to ensure that a user's data is secure.

CySky

Verified Discord Bot

CySky is an easy-to-use Discord Bot which provides the end user with helpful stats for Hypixel Skyblock. It is written in Python and uses the Hypixel API to retrieve data. It is currently out of development and is hosted on my servers. As of March 6th 2023, the bot is in 116 unique Discord servers!.
